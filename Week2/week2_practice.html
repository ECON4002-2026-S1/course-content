  • Calculate $\det(A_x)$, $\det(A_y)$, and $\det(A_z)$, where $A_i$ is $A$ with column $i$ replaced by $b$.

b <- c(3, 2, 4)

# Cramer's Rule: replace each column with b

Advanced Problem 11: Market Solver Function

Write an R function called solve_market that:

  • Takes demand intercept a, demand slope b, supply intercept c, and supply slope d as arguments
  • Forms the coefficient matrix and constant vector
  • Checks whether a unique solution exists (i.e., $\det \neq 0$)
  • Returns the equilibrium price and quantity, or an informative error message if no unique solution exists

Test with: $Q_d = 150 - 5P$, $Q_s = -30 + 10P$.

solve_market <- function(a, b, c, d) {
  # System: Q + b*P = a   (demand: Q = a - b*P)
  #         Q - d*P = c   (supply: Q = c + d*P)
  A <- matrix(c(1, b, 1, -d), nrow = 2, byrow = TRUE)
  rhs <- c(a, c)

  if (abs(det(A)) < 1e-10) {
    return(list(error = "No unique equilibrium — check your supply and demand parameters"))
  }

  eq <- solve(A, rhs)
  return(list(Q_star = eq[1], P_star = eq[2]))
}

# Test: Qd = 150 - 5P, Qs = -30 + 10P
result <- solve_market(a = 150, b = 5, c = -30, d = 10)
cat("P* =", result$P_star, "  Q* =", result$Q_star)

Advanced Problem 12: Sensitivity Analysis

Using the two-market model from Problem 7:

  • Write a function that calculates equilibrium prices given the parameters of the model.
  • Perform sensitivity analysis: how does $P_W^*$ change when the corn supply intercept changes from −20 to −10?
  • Create a plot showing how $P_W^*$ varies as the corn supply intercept ranges from −30 to 0.
two_market_eq <- function(corn_supply_intercept) {
  # Wheat: 5Pw - Pc = 90
  # Corn:  -Pw + 7Pc = 120 + corn_supply_intercept + 20
  #   (the corn equilibrium condition changes with supply intercept)
  #   Corn: Qc_d = Qc_s → 100 + Pw - 4Pc = corn_supply_intercept + 3Pc
  #                      → -Pw + 7Pc = 100 - corn_supply_intercept
  b_corn <- 100 - corn_supply_intercept

  A <- matrix(c(5, -1, -1, 7), nrow = 2, byrow = TRUE)
  b <- c(90, b_corn)

  prices <- solve(A, b)
  return(prices[1])  # Return Pw*
}

# Sensitivity: corn supply intercept ranging from -30 to 0
intercepts <- seq(-30, 0, by = 1)
Pw_values <- sapply(intercepts, two_market_eq)

plot(intercepts, Pw_values, type = "l", lwd = 2, col = "darkblue",
     xlab = "Corn Supply Intercept", ylab = "Wheat Price (Pw*)",
     main = "Sensitivity of Wheat Equilibrium Price")
